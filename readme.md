# SICOKRO - Sistem Informasi Yayasan Cokroaminoto

Sistem informasi terintegrasi untuk manajemen yayasan pendidikan.

## Fitur Utama
- Manajemen Tagihan & Pembayaran SPP
- Manajemen Aset Sekolah
- RKAS, RKT, RKJM
- Sumbangan & Bantuan
- Laporan Keuangan

## Instalasi
1. Clone repository
2. Import `sql/sicokro.sql` ke MySQL
3. Konfigurasi `app/config.php`
4. Akses via browser: `http://localhost/sicokro/public`
